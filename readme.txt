======= README ========

! Technology used
* node -v v16.20.0
* npm -v 8.19.4 
* Express JS for backend
* Nuxt 3 for frontend/dashboard

A. API integration ( ** Mandatory )

Folder used : Backend, Frontend

installation guide Backend
- cd backend
- npm install

installation guide Frontend
- cd frontend
- npm install
- npm run dev

B. Custom Dashboard with Drag-and-Drop ( ** Optional )

Folder used : Backend, Dashboard

installation guide Dashboard
- cd frontend
- npm install
- npm run dev

!!! Module B membutuhkan folder backend untuk mengambil data dari OpenWeatherMap
